# 短期予報解説資料/関連天気図等まとめサイト 総合操作マニュアル

## 1. サイトの概要
本サービスは、気象庁が発表する「短期予報解説資料」や、対応する時刻の各種天気図・衛星画像・数値予報図などの専門気象資料を、発表日時ごと、あるいは資料の種類ごとにまとめて効率よく閲覧・検証できるサービスです。
気象予報士試験等の高度な学習支援や、日々の実践的な気象解析（実況把握・予想の検証）をスムーズに行うことを目的に開発されています。解説資料の自動音声読み上げ機能、時系列・図版の拡大表示、加工元データの詳細確認など、実務や学習に役立つ多彩な機能を備えています。

---

## 2. 画面構成とサイドメニュー
画面左側に常設されているサイドメニューから、主要な各ページへいつでも容易にアクセスできます。

* **Home（トップページ）**: サービスの概要、更新情報、および基本的な使い方ガイドが表示されます。初めて利用する際の入り口となります。
* **Recent（直近の資料一覧）**: 直近一週間の発表日時一覧が時系列で表示されます。最新の気象データや直近の予報解説にダイレクトにアクセスできます。
* **Charts（各種図面一覧）**: 「短期予報解説資料」「実況天気図」「レーダー画像」「各種高層予想図」など、資料の種類（カテゴリ）ごとに分類された一覧ページです。特定の図面を1週間分の過去まで遡って比較・検証する際に最適です。
* **About（当サイトについて）**: 本サービスの開発背景（動機・目的・機能紹介）をはじめ、利用している出典情報（気象庁データの権利・取得元URL）やOSSライセンスに関する詳細情報が掲載されています。

## 3. 基本的な操作手順と詳細機能

### 3.1. Recent（直近の資料一覧）からのアクセス
1. 左側メニューの **[Recent]** をクリックします。
2. 「2026年07月22日 03時40分 発表」のように、直近の発表日時がリスト形式で表示されます。
3. 目的の発表日時をクリックすると、その日時に紐付けられた短期予報解説資料と、一連の対応天気図（実況・高層・レーダー等）が網羅された**日時毎の詳細ページ**に遷移します。

### 3.2. 詳細ページ（発表日時毎の閲覧ページ）の高度な活用
選択した日時の詳細ページでは、多角的な気象解析を行えるよう以下の機能が提供されています。

* **ページ内クイックジャンプ（目次）**: 長いページ内をスムーズに移動できるよう、上部に各資料への目次・リンクが配置されています。
* **時系列の切り替え（新旧ナビゲーション）**: 
  * 画面上下に配置されている `<<(new)` や `(old)>>` ボタンをクリックすることで、直前あるいは直後の発表日時データへシームレスに切り替えることができます。これにより、気象データの時間変化（時系列変化）を素早く追跡できます。
* **図面タイトルのクリック挙動（単体タブ表示）**:
  * 各図面項目の**タイトル部分をクリック**すると、その図面単体を大きく表示する専用の新しいタブ（または画面）が開き、じっくりと詳細を確認できます。
* **図面のクリック挙動（図表示モード）**:
  * 各図面項目の**図面をクリック**すると、ズーム、パン等が容易に行える図表示モードに切り替わります。特に画面の小さなスマートフォン等での図面確認に有効です。
* **短期解説資料の音声読み上げ機能**:
  * 画面下部に固定表示されるコントロールパネル（再生/一時停止ボタン、トラック表示 `TRACK : 1/8` など）を使用して、解説文の自動音声読み上げ機能を利用できます。
* **加工元データ詳細ページへの移動**:
  * 詳細ページの最下部に設置されている **「 データ詳細情報 」** のリンクをクリックすることで、当該発表日時の各資料のメタデータ（ダウンロード時刻（UTC）、ダウンロード元公式URL、ファイル形式、ファイルサイズなど）を詳細に確認できる専用ページへ移動します。
  * ※加工元データ詳細ページから元の天気ページへ戻る際は、ページ内の **「← 天気図ページに戻る」** リンクを使用します。

### 3.3. Charts（各種図面一覧）からのカテゴリ別アプローチ
1. 左側メニューの **[Charts]** をクリックします。
2. 「短期予報解説資料」「実況天気図（アジア太平洋域）」「レーダー画像」「極東850hPa気温・風、700hPa上昇流・湿度、500hPa気温予想図」「日本850hPa相当温位・風予想図」などのボタンが一覧表示されます。
3. 閲覧したい資料名を選択すると、その図面について過去一週間分の時系列サムネイルが並んだ専用ページが表示され、図をクリックすることで図表示モードに切り替わります。

### 3.4. 図面の拡大表示（図表示モード）の操作方法
1. Chartsページや詳細ページ内にある任意の天気図・画像サムネイルをクリックします。
2. 背景が暗転し、高解像度の画像が画面いっぱいに表示される **「図表示モード（モーダル/ビューア）」** に切り替わります。
3. **図表示モード中の主な操作**:
   * **前後の資料への切り替え**: 画面左右に表示される矢印（` < ` ` > `）のクリック、またはタッチデバイスでの左右フリック操作により、前後の図面に素早く切り替えられます。
   * **拡大・縮小・移動（ズーム＆パン）**: マウスホイールのスクロールやピンチイン・ピンチアウト動作による拡大・縮小、およびドラッグ操作による表示位置の自由な移動が可能です。細部の等圧線や前線の位置、数値を確認する際に極めて有効です。
   * **閉じる**: 画面右上等にある「×」ボタン、または背景部分をクリックすることで、元のページへスムーズに復帰できます。

## 4. Aboutページ・詳細情報と法的・技術的背景
左側メニューの **[About]** では、本サービスの透明性と信頼性を担保するための以下の情報が整理されています。

* **1週間以上前のデータ閲覧**:
  * 過去に発表された長期間のデータを蓄積・検索するためのアーカイブページです。
* **操作マニュアル**: 
  * 本ページへのリンクが記載されています。
* **出典情報・データ利用について**: 
  * 気象庁ホームページの公開情報および利用規約に基づき、引用・加工している各種データ（PDF、SVG、レーダー・衛星画像等）の正式な取得元URLや、データ加工プロセスの概要、取得パラメータの解説が記載されています。
* **オープンソースソフトウェア（OSS）ライセンス**: 
  * 本システムの構築・動作に使用されている各種OSSのライセンス条項（MITライセンス、ISCライセンス等）および関連するGitHubリポジトリへのリンクが明記されており、オープンな開発エコシステムに準拠しています。
