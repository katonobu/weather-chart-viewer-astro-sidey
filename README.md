# ページ構成予定
```
+- Homeページ(M)
|  +- 最新天気図 # 難しいか?
|  |  +- 最新天気図へのリンク  # 難しいか?
|  +- お知らせ
|  |  +- サービス停止/UI変更等のお知らせ
|  |  +- その他注意事項等
|  +- 概要
|  |  +- このサービスの概要
|  +- ここだけ読んでも使えます
|  |  +- Recentページ
|  |  +- 天気図ページ
|  |  +- 図面表示モード
|  +- 使い方詳細
|  |  +- 使い方詳細ページへのリンク
|
+- 使い方詳細ページ(未)
|  +- 各画面の内容/操作説明
|
+- Recentページ(M)
|  +- 直近1週間
|  |  +- 各発表日時毎の天気図ページへのリンク
|
+- Archiveページ(M)
|  +- 保存されている天気図
|  |  +- 各発表日時毎の天気図ページへのリンク
|
+- Aboutページ(M)
|  +- 出典情報へのリンク
|  +- ossライブラリライセンスへのリンク
|  +- ページ作成の経緯
|  +- 連絡先
|
+- 天気図ページ
|  +- 気象庁発表日時
|  +- 前後ページへのリンク
|  +- データ詳細ページへのリンク
|  +- ページ内目次
|  +- 各種取得データ
|  +- 短期予報解説資料読み上げ制御パネル
|
+- データ詳細情報ページ
|  +- 気象庁発表日時
|  +- 各データの取得日時、URL等
|
+- 出典情報ページ(未)
|  +- 各データの取得方法、URL等
|
+- ossライブラリライセンス表示ページ(未)
|  +- 各OSSライブラリのライセンス表示


```
